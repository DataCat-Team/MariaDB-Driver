## 🧶 MariaDB Driver Pack
This repository contains the DataCat Yarn for using MariaDB databases in DataCat.

### 📥 How to install this yarn
There are 3 different ways to install this yarn:

1. Open the DataCat Editor and install this yarn with the built-in **Yarn Explorer**.
2. Go on the [page on the marketplace](https://market.data-cat.de/plugin/mariadb-driver) and install it from there
3. Go to the [Jekins Side](https://ci.julianweinelt.de/view/all/job/DataCat-MariaDB/), download the latest .yarn and put it into your ``yarns`` folder.

### 🚀 How to use this yarn
This package provides the following components for DataCat:
- 📥 Download drivers for MariaDB directly in DataCat
- ⭐ Use MariaDB-specific features in the editor
- 🚀 Use Flow as your SQL Agent for your MariaDB instance(s) - ``Coming soon``
